================================================================================
BeltSGEContainer - Mod de compatibilité DayZ
================================================================================

Ajoute un slot sur la ceinture SGE "Stalker Sunrise Belt" pour y attacher
le container ON_AContainer1 du STALKER Containers Pack.

DÉPENDANCES (ordre de chargement) :
  1. STALKER Equipment
  2. STALKER Containers Pack
  3. BeltSGEContainer

UTILISATION :
  - Ceinture avec slot container : classe "BeltSGEContainer_Sunrise_Belt"
  - Container attachable à la ceinture : classe "BeltSGEContainer_AContainer1"

Addon SGE utilisé : SGE_ClearSky (PBO Sunrise du mod STALKER Equipment).

================================================================================
